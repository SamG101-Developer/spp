## AI

None of the compiler code in this repository was block-generated by AI. AI has been used for the following purposes:

- **Code review**: security and optimization
- **Code completion**: end of line code completion suggestions
- **Libraries**: learning how to use the `llvm` library
- **Writing the error messages**: will be rewritten by hand
- **Writing some unit tests**: pure laziness

## Implementation Stack

- Compiler: `GCC 17.0.0 (trunk)`
- IDE: `CLion 2026.1`
- Build System: `CMake 4.2.1` + `Ninja 1.11.1`
- Build Tools: `FetchContent` + `find_package`
- Package Manager: `FetchContent` + `find_package`
- Testing Framework: `GoogleTest (fork)`
- Style Guide: `Google C++ Style Guide (ish)`
- Project Structure: `Moduels (split): .ixx/.cpp`
- STL Strategy: `Raw STL`
