# Hello World

The "Hello World" program in S++ is a simple example that demonstrates the basic syntax and structure of the language.
It consists of a single function, `main`, which is the entry point of the program. The `main` function uses the `print`
function to output the string "Hello, World!" to the console.

Here is the code for the "Hello World" program in S++:

```spp
fun main(args: Vec[Str]) -> Void {
    std::io::print("Hello, World!")
}
```
